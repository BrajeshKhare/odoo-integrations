Featuring...

  • Create Ticket

Events:

Translators:

  • Make Request

  • Send Ticket with Desk

First steps

There's really only one thing you need to do: Map your models

  • Go to the Cenit IO Integrations menu.
  • Create a new Data type
    • Name it as you will.
    • Select the model you wish to send over to Desk (say 'Helpdesk').
    • Select the 'Communication' library
    • and the specific schema you want to use (say 'ticket.json').
    • On the Mappings tab set how you want your helpdesks to translate to desk
    • Save your work.

That's it! Everytime you create or modify a HelpDesk in Odoo, a ticket will be create on your Desk account.

For more on mapping check the Documentation tab of the Cenit IO Integrations Client.
