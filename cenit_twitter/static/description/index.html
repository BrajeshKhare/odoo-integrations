Setting up...

First things first: SignUp with Twitter

Go to https://apps.twitter.com and create one.

You can also use any you already have.

Make sure you grant all permissions for the app

Move to the Keys and Access Token sections and

Take note of your Consumer Key and Secret

Generate your Access Token and secret and also write them down

That's pretty much it

After installing the addon a form will let you set the data

Featuring...

Data definitions

Various Schemas that will allow you to map your Odoo models as Twitter's.

  • Tweets
  • Direct messages
  • Followers
  • Lists
  • List members
  • List subscribers
  • Friendships
  • Favourites
  • Blocks.

Set ups

Several Webhooks where you can send or get data.

  • Status Update
  • Status Retweets
  • Status Mentions Timeline
  • Status User Timeline
  • Status Home Timeline
  • Direct Messages New

A handful of Events and Translators to decide when and how perform transformations.

Events:

  • Direct Message on created_at
  • Tweet on created_at
  • Retweet

Translators:

  • Export Tweet
  • Make Tweet ID
  • Import Tweet
  • Export Direct Message
  • Import Direct Message

Out of the Box Flows so you can focus on what's really important.

  • Tweet Send
  • Status Retweet
  • Direct Message Send
