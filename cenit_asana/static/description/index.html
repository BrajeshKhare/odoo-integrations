Setting up...

First things first: SignUp with Asana

All we need is your personal token.

Go to My Profile Settings. Move to Apps section, generate a new personal access token and take note of this.

That's pretty much it

After installing the addon, a form will let you set the data

Featuring...

Data definitions

Various Schemas that will allow you to map your Odoo models as Asana's.

  • Users
  • Projects
  • Tasks
  • Teams

Set ups

Several Webhooks where you can send or get data.

  • List Projects
  • List Tasks on Project
  • Create Project
  • Create Task
  • List users

A handful of Events and Translators to decide when and how perform transformations.

Events:

  • Task create_at
  • Project create_at

Translators:

  • Make Model ID
  • Export Model
  • Import List

Out of the Box Flows so you can focus on what's really important.

  • Asana API -> Create Project
  • Asana API -> Create Task

First steps

There's really only one thing you need to do: Map your models

  • Go to the Cenit IO Integrations menu.
  • Create a new Data type
    • Name it as you will.
    • Select the model you wish to send over to Asana (say 'Project').
    • Select the 'Asana' library
    • and the specific schema you want to use (say 'project.json').
    • On the Mappings tab set how you want your odoo's projects to translate to asana's projects
    • Save your work.

That's it! Everytime you create or modify a Project in Odoo, a project will be create on your Asana account.

For more on mapping check the Documentation tab of the Cenit IO Integrations Client.
